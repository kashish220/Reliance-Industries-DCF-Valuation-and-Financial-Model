# 📊 Reliance Industries Ltd | Financial Model & DCF Valuation 💼

This project presents a comprehensive Financial Model and DCF Valuation of Reliance Industries Ltd developed in Microsoft Excel. The model follows industry-standard practices used in Equity Research, Investment Banking, Corporate Finance, and Valuation Analysis.

The objective of this project is to analyze the company's historical performance, forecast future financials, estimate intrinsic value using Discounted Cash Flow (DCF) methodology, and evaluate overall investment attractiveness.
